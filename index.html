Overview

The Lightwheel Factory Manipulation Task Dataset is a structured benchmark for industrial manipulation and robot evaluation. It follows a documentation-centered presentation similar to the public Lightwheel dataset page, while shifting the content toward factory assembly, transport, inspection, and sorting workflows.

What it covers

  • Task coverage: pick, assembly, transfer, and conveyor sorting templates.
  • Multi-robot: UR5, UR10, and Franka Panda with task-specific compatibility.
  • Scene diversity: assembly line, warehouse storage, and quality inspection stations.

Why this format

  • Docs-first layout: easier to scan than a marketing landing page.
  • Consistent metadata: unified task templates and structured success predicates.
  • Deployment-friendly: static HTML and JSON, ready for GitHub Pages or other free hosting.

Dataset Format

The dataset follows a structured episode schema with scene metadata, constraints, and trajectory fields for training and evaluation pipelines.

{
  "episode_id": "factory_pick_001",
  "task_id": "pick_part_from_bin",
  "scene": {
    "scene_id": "factory_assembly_001",
    "scene_type": "assembly_line",
    "layout_seed": 42
  },
  "factory_metadata": {
    "cycle_time_requirement": 5.0,
    "safety_constraints": ["no_collision", "force_limit_30N"],
    "quality_thresholds": { "position_error": 0.005 }
  },
  "trajectory": {
    "observations": { "images": {}, "state": [] },
    "actions": [],
    "success": true
  }
}

Dataset Composition

Composition Summary

  • Templates: logic-based task definitions with success predicates and factory constraints.
  • Assets: workbenches, bins, conveyors, storage shelves, and inspection stations.
  • Procedural variation: scene tags, randomization fields, and layout seeds for diversity.

Recommended Uses

  • Imitation learning and policy finetuning for industrial tasks.
  • Cross-robot transfer evaluation across collaborative and industrial arms.
  • Benchmarking perception, scene grounding, and constrained manipulation planning.
