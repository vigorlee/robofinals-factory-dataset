Lightwheel Factory Manipulation Task Dataset

The Lightwheel Factory Manipulation Task Dataset is a large-scale, open repository of robot demonstration data for imitation learning and manipulation benchmarking, collected in diverse simulated factory environments. The full dataset materials are available in this published web page and the linked GitHub repository.

Overview

  • Task Coverage: factory pick, assembly, transfer, and conveyor sorting tasks in one benchmark page, spanning tabletop, line-side, and station-to-station manipulation.
  • Multi-robot: data is organized around three robot platforms: UR5, UR10, and Franka Panda, with per-task compatibility metadata.
  • Scale: 2,500 demonstration episodes are included in the current dataset snapshot, collected across multiple tasks, scenes, and robot pairings.
  • Diversity: episodes are gathered in realistic factory layouts with interactive fixtures, conveyors, storage zones, and inspection stations.
  • Standardization: all data follows a structured task schema and scene metadata layout, making filtering, loading, and benchmarking easier.

Data Structure & Access

The dataset follows a structured episode format with scene descriptors, factory-specific constraints, and trajectory payloads suited for training and benchmarking. It is packaged as a static web-friendly dataset description with JSON metadata and documentation assets. 

{
  "episode_id": "factory_pick_001",
  "task_id": "pick_part_from_bin",
  "scene": {
    "scene_id": "factory_assembly_001",
    "scene_type": "assembly_line",
    "layout_seed": 42
  },
  "factory_metadata": {
    "cycle_time_requirement": 5.0,
    "safety_constraints": ["no_collision", "force_limit_30N"],
    "quality_thresholds": { "position_error": 0.005 }
  },
  "trajectory": {
    "observations": {
      "images": {},
      "state": []
    },
    "actions": [],
    "success": true
  }
}

Applications

  • Imitation Learning: policy training for industrial pick-and-place, transfer, insertion, and sorting tasks.
  • Cross-Robot Evaluation: benchmarking task transfer between collaborative and industrial robot platforms.
  • Perception and Grounding: testing scene understanding under realistic factory clutter and layout changes.
  • Task Benchmarking: standardized episode structures support repeatable experiments and leaderboard-style comparisons.
